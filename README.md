# Automated Sports Market Settlement System

A Python project I built to better understand how prediction 
market settlement works.

## Files

- `settlement_system.py` — the main settlement script
- `capstone_log.txt` — a sample log from a real run
- `manual_review.csv` — sample manual review export

## Note

Scores are simulated for demonstration purposes. 
In production, these would come from a live data feed.
